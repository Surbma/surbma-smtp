== Installation ==

1. Upload `surbma-smtp` folder to the `/wp-content/plugins/` directory
2. Activate the Surbma - SMTP plugin through the 'Plugins' menu in WordPress
3. Set every global variable in the wp-config.php file.
3. That's it. :)
